Cat Boost

An interesting little automation script.

View Source Code on GitHub

Quick Start

Step 1: Install "Bongo Cat" from Steam.
Step 2: Download AutoHotkey v2.0.
Step 3: Run cat Boost.ahk.
Step 4: Press F1 in-game to toggle the script.

How it Works

To play the "Bongo Cat" game, you can download it from Steam. This is a simple script to speed up the accumulation of hits in the game Bongo Cat.

Each hit grants a 50x multiplier to the hit count, but this only applies to hitting the A through Z keys. A simple delay is included to prevent the game from detecting hits too quickly and not counting them. A section is also included to address the issue of the Shift key.

To try it out, press F1 to start the game!
